# 🚀 Portfolio

Portafolio personal desarrollado con **Astro**, enfocado en rendimiento, simplicidad y experiencia de usuario.

## 🧑‍💻 Sobre el proyecto

Este proyecto presenta mi experiencia, habilidades y proyectos como desarrollador, utilizando una arquitectura moderna y componentes reutilizables.

## 🛠️ Stack

- Astro  
- HTML / CSS  
- JavaScript / TypeScript  

## ⚡ Características

- Diseño moderno y responsive  
- Alto rendimiento (SSR / SSG con Astro)  
- Estructura modular y escalable  
- Fácil de mantener y extender
